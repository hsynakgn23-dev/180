Place custom mark badges in this folder.

Supported formats:
- `svg`
- `png`
- `jpg`
- `jpeg`
- `webp`
- `avif`

Naming convention:
- Use the mark `id` as file name.
- Example: `first_mark.svg`, `genre_nomad.png`

Optional override:
- In `src/data/marksData.ts`, set `badgeAsset` on a mark if file name is different.
